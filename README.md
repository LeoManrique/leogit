# leogit

An easy visual way to use Git without losing control, built in Go. [WIP]

## Target user

- Myself (opinionated design)
- New developers learning to use Git
- Experienced Git users who like visual feedback

## What it does (eventually)

- Clean layout with Changes and History tabs
- AI commit messages
- Embedded terminal at repo root
- Pull request workflows via `gh` CLI
- Branch management
- Background auto-fetch with ahead/behind indicators
- Syntax-highlighted diffs
- Cross-platform TOML configuration

## Requirements

- Go 1.21+
- `git` and `gh` in `$PATH`
